Employee Directory

The Employee Directory application was designed to facilitate the search for company employees. It offers a user-friendly interface that allows users to easily locate employee profiles based on department and name.

Live Link

Project Overview

For this capstone project, I was tasked with creating an employee directory for a growing company. The directory comprises four departments: Leadership, Sales, Customer Service, and Product.

To begin, I received a dataset containing employee information such as name, department, and bio. Additionally, I was provided with a Figma file that outlined the design specifications. My responsibility was to build the entire design from scratch, incorporating the Figma design specifications and graphics.

One of the key requirements was to dynamically render each employee on the page using JavaScript and the provided employee data. Furthermore, I was required to implement a filtering feature that allows users to sort employees based on department.

In order to challenge myself and enhance the project, I decided to utilize React components for its development. Additionally, I incorporated a search feature, enabling users to easily search for employees by their names.

Considering the importance of responsiveness, I took the initiative to make the project compatible with various devices. By utilizing CSS grid, I ensured that the directory would appear visually appealing and well-organized across screens of all sizes.

I encountered a significant challenge during this project: implementing real-time rendering of employee search results as the user types. This required extensive research and experimentation to find an optimal solution. Eventually, I discovered the slice() method, which proved to be the perfect tool for the task.
