Salesboard

This Salesboard web application is an all-inclusive toolkit that empowers employees to effortlessly track their sales and closely monitor their performance hassle-free.

Live Link

Project Overview

For this capstone project, I undertook the exciting challenge of developing a cutting-edge mock sales tracking platform to replace the old and outdated whiteboard tracking method.

The primary objective of this platform is to streamline sales tracking and provide a user-friendly interface for the sales team. I was given the data file along with design specications via Figma.

My task involved implementing a functionality where, upon clicking either of the two product buttons, the user would be awarded badges for their live sales and live achievements based on their sales performance.

I also designed a system to grant a live achievements badge for significant milestones, such as their first and fifteenth sales, as well as when employee achieve a total revenue of $2500.

To challenge myself further, I created a sign in page that needs a valid username and password before gaining access to the Salesboard application. To get some practice with local storage, I decided to add a feature that enables users to save their sales data. I thought it would also be neat to create a toggle feature that toggles from dark to light mode so I included that. A sign out feature was also added. Furthermore, I incorporated a sign-out feature to provide users with an intuitive way to log out.

It wasn't required for me to create this project using React, but I decided to use it to get some more practice.
